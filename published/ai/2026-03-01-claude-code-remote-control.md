## 📝 正文内容

通勤路上突然想到一个bug怎么改，但电脑不在身边？

以前只能干着急，现在 Claude Code 的 Remote Control 功能帮你搞定。

【怎么用？3步搞定】

1. 电脑端启动
终端输入 /rc 或 claude remote-control，弹出二维码或链接

2. 手机扫码
用手机浏览器打开链接，iOS/Android 都支持

3. 开始远程操作
手机变成 Claude Code 的远程窗口，可以发指令、看进度、确认改动

【几个实用场景】

- 通勤时间：地铁上让 Claude 帮你写代码
- 摸鱼间隙：假装看手机，实际在改bug
- 躺平时刻：沙发上也能完成工作
- 外出应急：不在电脑前也能处理

【注意事项】

- 电脑需保持开机，Claude Code 需运行中
- 网络要稳定，建议 WiFi
- 目前仅 Max 订阅用户可用

【我的真实体验】

用了两周，最大的感受是终于不用随时背着电脑了。以前周末出门还惦记工作，现在手机就能远程操作，随时处理紧急需求。
